Notes

Additive Identity - is zero because when add zero to a value the value remains the same
Multiplicative Identity - is one because when multiply a value by one the value remains the same
Inverse - result is that you get the identity
Additive Inverse - add its opposite so results in zero ex: a + (-a) = 0
Multiplicative Inverse - of a number and its reciprical - always get one ex: a * 1/a = 1
Commutative Property - The order of operation does not matter - addition and multiplication are commutative, subtraction and division are not.
Associative Property - When performing the same operation on three numbers, the order/grouping of parathesis does not matter.
Distributive Property - When a value is multiplied by a sum would get the same value if multiply each value seperately and then add products ex: x(a + b) = xa + ab
