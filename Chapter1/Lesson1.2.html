Equation - a statement in math in which two expressions are equal - it contains an equal sign
Steps to Solve a Linear Equation for one Variable:
  1. Simplify - simplify each side of the equation
    1. use the distributive property where possible to remove parenthesis
    2. combine like terms on the same side as the equal sign if possible
  2. Bring variable terms to one side - if there are variables on both sides - eliminate a variable term from one side by adding its opposite to both sides
    1. if one side has more terms eliminate from that side
    2. otherwise, eliminate the term with the smaller coefficient
  3. Isolate the variable - use the reverse order of operation
    1. add the opposite (additive inverse) of the constant term to both sides
    2. divide both sides (or multiply both sides by reciprical) by the variables terms' coefficient
  4. Check! substitute the variable with your solution
