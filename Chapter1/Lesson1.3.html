Notes

Inequality - is a statement that one expression is compared to, but not equal to another
When Multiplying/Dividing both sides by a Negative number - reverse the inequality symbol
Graphing an inequality:
  1. On the number line, find the value that the variable is being compared to
  2. If the inequality symbol is > or < draw an open circle at that value, which means, thst number is not included in the solution set. If the inequality sybol is >= or <= draw a closed circle at that value which means it is included in the solution set
  3. if the variable is > or >= draw the arrow to the right, if the variable is < or <= draw the arrow to the left
