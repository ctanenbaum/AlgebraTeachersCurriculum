Notes

Line - consists of a set of points. Each are presented by an ordered pair
Ordered Pair - states its x value and y value
We can create a table made up of our ordered pairs:
x y
-1
0
1
If y is equal to a constant - the line is parallel to the x-axis
If x is equal to a constant - the line is parallel to the y-axis
Determining whether a point is on the line - substitute the x value and y value for the variables x and y in the equation and then check if these values make the equation true

Classwork

  1. Write the equation of a line parallel to the y-axis but 9 units to the right of it.
  2. Write the equation of a line parallel to the x-axis but 10 units below it.
  3. Does the line whose equation is 2y + 6 = 4x contain the point (1, -1)?
  4. Does the point (3,7) lie on the line whose equation is y = 3x - 2?
  5. Does the line whose equation is y = 4x pass through the origin?
  6. Determine if the ordered pair (-4,3) is a solution of 4x - y = -13

Homework

  1. Write the equation of a line that lies on the y-axis
  2. Write the equation of the line parallel to the x-axis but 1 unit above it
  3. Does the line whose equation is y = 1/2x + 5 contain the point (4,9)?
  4. Does the point (-2,-4) lie on the line whose equation is 2y - 3x = -2?
  5. The graph of the equation 2x + 6y = 4 passes through the point (x,-2). What is the value of x?
